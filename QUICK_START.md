
# Quick Start

1. Unzip the folder.
2. Open Terminal inside the folder.
3. Run:

```bash
pip3 install -r requirements.txt
streamlit run app.py
```

4. Upload `sample_inventory_v3.csv`.
5. Try these tabs:
   - Post This Item
   - Social Scheduler
   - Weekly Marketing Plan
   - Customers
   - Wishlist Matching
   - Reports
