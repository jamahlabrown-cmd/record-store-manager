# Quick Start

1. Upload app.py and requirements.txt to GitHub.
2. Deploy/reboot Streamlit.
3. Set ADMIN_PASSWORD in Streamlit Secrets.
4. Register a buyer.
5. Submit a seller application.
6. Admin approves seller.
7. Seller logs in with email and access code.
8. Seller creates policies and products.
9. Buyer can buy/message/report.
10. Admin reviews only flagged issues.
