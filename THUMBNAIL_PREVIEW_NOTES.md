# House Of Wax V9 Thumbnail Previews

This version adds thumbnail previews for saved internet media links.

## What works

- Direct image URLs can render as thumbnails.
- YouTube links automatically display a thumbnail image.
- Direct video URLs can show an embedded video or a thumbnail if you provide one.
- You can paste an optional `thumbnail_url` when saving an internet media link.

## Best practice

For the best customer experience:

- use your own photos whenever possible
- use official YouTube links for videos
- use direct image links for pictures
- add a thumbnail URL for any non-direct video/image page so customers can preview it first
